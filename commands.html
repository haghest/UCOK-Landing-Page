Bot Commands

Description : Add a Filter to the Filters
Aliases : [+addfilters] [+add] [+addf]
Usage : addfilter {Filter1} {Filter2}

Description : Sets a custom Bassboost with Gain!
Aliases : [+bassboost] [+bb] [+bass] [+custombass] [+cbassboost] [+cbass] [+cbb] [+custombb]
Usage : custombassboost {Gain (0-20)}

Description : Changes the Speed of the Song
Aliases : [+customspeed] [+changespeed] [+cspeed]
Usage : speed {speedamount (0 - 20)}

Description : List all active and possible Filters!
Aliases : [+listfilter] [+listfilters] [+allfilters]
Usage : filters

Description : Removes a Filter from the Queue
Aliases : [+removefilters] [+remove] [+removef]
Usage : removefilter {Filter1 Filter2}

Description : Sets (Overwrides) all Filters
Aliases : [+setfilters] [+set] [+setf]
Usage : setfilter {Filter1 Filter2}
